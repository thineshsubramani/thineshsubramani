My Open-Source Projects

Note: All these are my open-sourced versions. Previously, code was built for organizational needs; here you see the curated, working public versions.

Projects

  • GitHub Runner Prometheus Exporter – Exposes GitHub Actions runner metrics via Prometheus by scraping local logs, ideal for monitoring multiple self-hosted runners at scale.
  • Sheet CLI – Fast Go-based CLI tool that fetches and displays syntax snippets for languages like Go and Python from a remote YAML backend, with caching, alias support, and colorized output.
  • Syskit – Toolkit for managing system configurations and automating tasks across multiple servers, streamlining infrastructure management.
  • Repo Governance – Governance-as-code toolkit to discover, validate, and enforce tags, workflows, README standards, and policies across all GitHub repositories.
  • Syslite – Minimal system information extractor in Go, designed to provide essential system details like hostname and uname in a lightweight manner.
  • Sync SSH ID – CLI tool to inject, update, or delete SSH public keys across multiple hosts, with YAML inventory and known_hosts fingerprint handling.
  • Network Debug Script – Script to diagnose and troubleshoot network issues by performing multiple checks and providing actionable insights.
